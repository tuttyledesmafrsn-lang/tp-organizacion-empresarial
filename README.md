## Escenario seleccionado

Escenario B - Análisis de Ventas de una Pequeña Empresa

## Descripción

Este proyecto fue desarrollado para la materia Organización Empresarial de la Tecnicatura Universitaria en Programación (UTN).

El objetivo es analizar un conjunto de datos de ventas mediante Python, utilizando Git, GitHub y Jira para la gestión del proyecto y el control de versiones.

## Estructura del repositorio

* datos/: contiene el dataset utilizado.
* scripts/: contiene el código fuente.
* resultados/: contiene gráficos y resultados generados.

## Herramientas utilizadas

* Git
* GitHub
* Jira
* Python
* Google Colab

## Ejecución

1. Abrir el script de análisis.
2. Ejecutar el código.
3. Revisar los resultados generados en la carpeta resultados.
